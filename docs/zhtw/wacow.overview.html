1.1 簡介

Wacow Framework 是基於 Zend Framework 上，一個用來快速產生擁有後台管理的 CMS 框架；其全名為 WAbow Constructing Outstanding Web application 。

Wacow Framework 主要目的是提供目前專案常用的功能，並節省我們開發後台的時間。
