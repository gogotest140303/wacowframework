1.2 Installation (安裝)

1.2.1 系統需求

Web Server

  • Apache 2 以上版本並加掛 mod_rewrite 模組

PHP

  • PHP 5.2.1 以上版本
  • Zend Framework 1.0.1 以上版本

Database

  • MySQL 4.1 以上版本

1.2.2 安裝資料庫

Wacow 目前提供基本的 SQL Schema 供開發者使用，請利用您慣用的 MySQL Client 匯入 /app/sql/schema.sql 。

1.2.3 新增設定檔

  1. 將 /app/config/config.sample.ini 複製為 config.ini 。

  2. 修改 config.ini 中的項目，例如資料庫連線等。
